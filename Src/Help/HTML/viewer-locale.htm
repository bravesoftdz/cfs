Locale Viewer

This window displays information about the locale specified by the CF_LOCALE clipboard format.

A subset of the locale information is displayed, as follows:

  • Language identifier in hex digits.
  • Full localised name of language.
  • Full English name of language from ISO639.
  • Abbreviated name of language, created by taking the 2-letter language abbreviation from ISO639 and adding a third letter to indicate the sublanguage if required.
  • Country code based on international phone codes.
  • Full localised name of country.
  • Full English name of country.
  • Abbreviated name of country per ISO3166.
  • Original equipment manufacturer code page associated with country.

There is a context menu that appears when any of the text boxes that contain locale information is right-clicked. This menu gives the option to select and copy the text in CF_TEXT format.
