HTML Clip Viewer

This window displays information about the "HTML Format" clipboard format.

The "HTML Format" format is often used when content is copied from a web browser to the clipboard. The format is text based and not only supplies the copied HTML but also provides some data about the copied HTML. The information provided includes the context of the copied HTML containing all required surrounding tags, the fragment actually copied, information about the selection (optional) and the source URL. The version of the clip format is also displayed.

Details of the various pieces of provided information are displayed at the top of the window and the HTML is rendered in the lower half of the window.

To view the underlying data right click "HTML Format" in the main display and select View As Text from the popup menu.

A context menu is provided to enable the content of the view to be selected and copied to the clipboard.

Note: Raw HTML, with no additional clip data, is not displayed in this viewer – see the related HTML Viewer.
