Text Viewer

This window displays ANSI and Unicode plain text clipboard formats such as CF_TEXT, CF_OEMTEXT, CF_UNICODETEXT.

The viewer as also available as a secondary viewer for some clipboard formats such as "Rich Text Format" that are composed of structured text. In this case the underlying "source code" is displayed, while any primary viewer will render the source code.

Text can optionally be word wrapped by checking the Word Wrap check box that appears below the text display. This setting is remembered by the program.

There is a context menu that enables text to be selected and copied to the clipboard.
