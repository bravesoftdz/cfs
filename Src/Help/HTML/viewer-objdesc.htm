Object & Link Source Descriptor Viewer

This window displays information about a link source descriptor or an object descriptor described by the "Object Descriptor" and "Link Source Descriptor" clipboard formats. The following information is displayed:

  • CLSID – GUID representing the object or link source.
  • The draw aspect of the object or link source: DVASPECT_CONTENT, DVASPECT_THUMBNAIL, DVASPECT_ICON, DVASPECT_DOCPRINT or a comment that the source application did not draw the object.
  • The extent (size) of the object if specified.
  • Status flags in hex which are a bitmask of the Windows OLEMISC enumeration.
  • Full user type name of the object or link source if provided.
  • Name that identifies the display name of the data source.

There is a context menu that appears when any of the text boxes that contain the provided information is right-clicked. This menu gives the option to select and copy the text in CF_TEXT format.
