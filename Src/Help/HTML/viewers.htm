Viewers

Various clipboard formats can be displayed by Clipboard Format Spy. To display a clipboard format right click the desired format in the main window to display the viewer pop-up menu. The desired viewer can then be selected.

Some formats have dedicated primary viewers that can also be displayed by double clicking the format in the main display. The binary data of all memory based formats can be viewed in a hex viewer. Formats that are based on plain text can also be viewed as text.

Available viewers are:

  • Text viewer: Used to display plain text or the raw text of structured text such as HTML or RTF. Can display Ansi, Unicode or OEM text.
  • Binary viewer: Used to display a hex representation of the binary data of any memory based clipboard format.
  • Rich Text viewer: Used to display a representation of rich text as stored in the "Rich Text Format" and "Rich Text Format Without Objects" clipboard formats.
  • HTML viewer: Used to display partial and complete HTML documents stored by, for example, the "text/html" clipboard format
  • HTML clip viewer: Used to display the formatted HTML and properties of an HTML clip stored in the "HTML Format" clipboard format.
  • Graphic viewer: Used to display metafile and bitmap clipboard formats.
  • File list viewer: Displays clipboard formats that contain a list of files, e.g. CF_HDROP and "Shell IDList Array"
  • Filename viewer: Displays the name of a file or folder copied placed on the clipboard in the "FileName" and "FileNameW" formats.
  • URL viewer: Displays the name of a URL placed on the clipboard using the "UniformResourceLocator" and "UniformResourceLocatorW" formats.
  • Locale viewer: Used to display selected information about a locale stored in the CF_LOCALE clipboard format.
  • Object & Link Source Descriptor viewer: Used to display information about objects and link sources.
