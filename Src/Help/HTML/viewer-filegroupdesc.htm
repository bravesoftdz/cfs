File Group Descriptor Viewer

This window displays information about a group of (one or more) files copied using the "FileGroupDescriptor" and "FileGroupDescriptorW" clipboard formats. Supported information types are:

  • CLSID – a GUID identifying the file.
  • Size of file's icon and its co-ordinates on screen.
  • File attributes
  • Creation, last access and last modification times of the file.
  • Size of the file.

Not all types of information may be provided. When absent the information is omitted from the list.

A context menu is provided to enable a list of the file names in the group to be copied to the clipboard as CF_TEXT. Each file name is separated by a CRLF pair.
