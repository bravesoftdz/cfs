Overview

This program displays the formats currently recorded on the clipboard. It automatically updates the information displayed as the contents of the clipboard change. For certain supported formats you can also view the clipboard contents. The contents of the clipboard can be deleted.

For each format displayed on the clipboard, the following information is displayed in the main window:

  • Code – this is the identifier used to represent the clipboard format. It is displayed in hexadecimal.
  • Description – this is the description of the clipboard format. Symbolic constants are displayed for the Windows standard clipboard formats.
  • Size – this is the size of the data on the clipboard. No value is displayed if the clipboard contents are not stored in global memory.

The display can be sorted in ascending or descending order on any column. Simply click a column header to sort by that column. Clicking twice reverses the sort order. The display columns can be resized by dragging the edge of the column headers.

To view a clipboard format select it in the display and right-click to display a pop-up menu. Select the required viewer from the menu. If the format can't be viewed "no viewer" will be displayed in the menu. More info.

The file name of the process that owns the clipboard is displayed in the status bar. This is the process that last copied data to, or cleared, the clipboard.

The window can be resized and the program remembers the size and position information between executions.

There is a a toolbar which provides the following functions:

Clear the clipboard.
Display this help file.
Display an about box containing copyright and version information.
Exit the program.
