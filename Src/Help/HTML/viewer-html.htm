HTML Viewer

This window renders HTML that has been copied to the clipboard using the "text/html" format used by Firefox and the "HTML (HyperText Markup Language)" format used by OpenOffice.

Clipboard Format Spy provides its own default cascading style sheet to style the HTML, so the rendered HTML may have a different appearance to the original.

Note: this is not the viewer used for the "HTML Format" clipboard format – this specialised format provides further information alongside the HTML. This format is displayed in the HTML Clip Viewer.

A context menu is provided to enable the content of the view to be selected and copied to the clipboard.
