Binary Viewer

This window displays binary information about any clipboard format that is represented in global memory. Supported formats will have a size entry in the main window. This binary viewer is always a secondary viewer.

The viewer shows 16 bytes of data per line, with each line divided into three sections:

  • Offset – this is the offset of the start of the "line" of 16 bytes from the start of the data. The offset is expressed in hex digits and appears on the left of the display.
  • The value of each of the 16 bytes is displayed in the centre of the display in hex.
  • The right hand side of the display shows the printable bytes of data as text. Unprintable characters are represented as dots.

The display can be scrolled up and down and left to right using the scroll bars or by using the following keys:

Key Action
End Scrolls right to display the end of the line.
Ctrl+End Moves to the end of the document.
Home Scrolls left to display the start of the line.
Ctrl+Home Moved to the start of the document.
Left Scrolls left.
Right Scrolls right.
Down Scrolls down one line.
PgDn Scrolls down one screen.
Up Scrolls up one line.
PgUp Scrolls up one screen.

There is no context menu available in this window.
