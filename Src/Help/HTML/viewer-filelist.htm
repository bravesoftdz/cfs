File List Viewer

This window displays lists of files that have been provided via the CF_HDROP and "Shell IDList Array" clipboard formats.

In the case of CF_HDROP the display lists files that have been copied to the clipboard whilst for "Shell IDList Array" the display shows both a root folder and a list of contained files. The display distinguishes between files and folders – different icons are used.

Right clicking in the main window displays a popup menu. The items on the menu depend on the context.

  • A Copy File List menu item is always available. This causes a list of files to be copied to the clipboard as text, with each file or folder name separated by CRLF.
  • If a folder is right clicked then an Explore default menu item is also provided. Clicking this item causes Explorer to be opened with the folder selected.
  • When a file entry is right clicked the default option changes to Open and selecting the option causes the program to attempt to open the file in any associated application. If there is no such application an error message is displayed.
